# ⚡ Really Fast Start

**Really Fast Start** gives you a strong early-game builder loadout so you can skip the slow first steps and get straight into building your factory.

Start with **Power Armor MK2**, a filled equipment grid, faster personal construction bots, repair packs, and enough movement speed and roboport coverage to make early expansion feel smooth.

No long burner phase.  
No slow walking everywhere.  
No waiting forever for basic construction.

Just spawn in and start building.

---

## ⚙️ What It Does

Really Fast Start gives every new player a ready-to-use personal builder kit.

The mod equips you with **Power Armor MK2** and fills the armor grid with useful equipment like reactors, exoskeletons, personal roboports, batteries, and shields.

It also gives you **50 custom Really Fast Start Construction Robots**. These bots are faster than normal construction robots, but they are their own custom robot type, so the mod does **not** change the speed of every robot in the game.

That means your starter bots feel quick and useful, while the rest of the game stays clean and predictable.

## 🎒 Starter Kit

New players receive a strong builder-focused loadout.

### Armor

* 1× Power Armor MK2

### Equipment Grid

* 2× Portable Reactor / Fission Reactor, depending on available prototypes
* 4× Exoskeleton
* 4× Personal Roboport MK2 if available
* Falls back to normal Personal Roboports if MK2 roboports are not available
* 6× Battery MK3 if available
* Falls back to Battery MK2 if MK3 batteries are not available
* 2× Energy Shield MK2

### Inventory

* 50× Really Fast Start Construction Robots
* 100× Repair Packs

---

## 🤖 Really Fast Start Construction Robots

Really Fast Start includes its own custom construction robot type.

These bots are designed for the starter kit and are faster than normal construction robots.

This keeps the mod clean and balanced:

| Robot Type | Behaviour |
| ---------- | --------- |
| Really Fast Start Construction Robot | Faster starter bot from this mod |
| Vanilla Construction Robot | Unchanged |
| Vanilla Logistic Robot | Unchanged |

The mod does **not** apply a game-wide robot speed bonus.

Only the bots provided by Really Fast Start are faster.

## 🛠️ Admin Command

If you need to give the kit again, run:

```txt
/really-fast-start
```

Admins can also give the kit to another player:

```txt
/really-fast-start PlayerName
```

The old command is still included as a legacy alias:

```txt
/fast-start-kit
```

This is useful for multiplayer servers, testing worlds, or if someone joins after the starting kit has already been handed out.

## 📌 Design Goal

Really Fast Start is made for players who enjoy Factorio but do not want to repeat the slowest part of the early game every time.

It is especially useful for:

* New saves where you want to build quickly
* Multiplayer servers
* Mod testing
* Blueprint testing
* Large base planning
* Players who prefer building over early-game walking and handcrafting

The aim is to make the start feel powerful without rewriting the whole game.

You still need to build your factory, automate production, research technology, and expand properly.

This mod just gets you moving faster from the first minute.

---

## 🔗 Compatibility

Really Fast Start is designed to be simple and safe.

It should work with:

* Vanilla Factorio 2.0
* Space Age
* Multiplayer servers
* Most overhaul mods
* Mods that add higher-tier batteries or roboports

If MK3 batteries are not available, the mod automatically uses MK2 batteries instead.

If Personal Roboport MK2 is not available, the mod can fall back to normal Personal Roboport equipment.

---

## 🧪 Behaviour Notes

* The kit is given when a player joins or starts
* Existing normal construction robots are not changed
* Existing logistic robots are not changed
* Robot speed research is not changed
* The mod does not unlock all technology
* The mod does not change recipes
* The mod does not change enemy balance
* The mod is meant as a clean starting boost, not a full creative-mode replacement
