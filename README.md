# Really Fast Start

Really Fast Start gives each player a clean builder start for Factorio 2.0.

## Starter kit

- Power Armor MK2 equipped automatically
- Filled armor grid:
  - 2x portable/fission reactor equipment, using available prototype fallback
  - 4x exoskeleton equipment
  - 4x personal roboport MK2 equipment, falling back to normal personal roboports
  - 6x battery MK3/MK2 equipment, falling back safely
  - 2x energy shield MK2 equipment
- 50 Really Fast Start construction robots
- 100 repair packs

## Robot speed

Only the custom Really Fast Start construction robots are 3x faster. Vanilla construction robots and logistic robots are not changed.

## Commands

```txt
/really-fast-start
```

Admins can give the kit to another player:

```txt
/really-fast-start PlayerName
```

The old command still works as a legacy alias:

```txt
/fast-start-kit
```
